------------INFOS--------------
Um das Spiel zu starten, müsst 
ihr auf die Pen Simulator v1.0
clicken. Falls da eine Warnung
kommen sollte, ignoriert diese.
liegt daran, das es eine 
unbekannte.exe datei ist.

---------WARNUNG--------------------
Das Spiel funktioniert nur in
neueren Browsern, die localstorage
unterstützen. Wenn ihr es nicht wisst,
drückt im Browser F12 und schreibt in
der Konsole localstorage. Wenn da ein 
Fehler kommt, geht es in diesem Browser
nicht.
